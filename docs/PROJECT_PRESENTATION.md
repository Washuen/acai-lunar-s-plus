# PROJECT_PRESENTATION.md — Apresentação para Portfólio

## Nome

**Açaí Lunar S+ Super**

## Pitch curto

Plataforma fullstack premium de delivery de açaí com cardápio visual, carrinho, checkout real, área do cliente, painel administrativo, controle de produtos, cupons, estoque, relatórios, autenticação JWT, PostgreSQL, Prisma e deploy preparado.

## Pitch completo

O Açaí Lunar S+ Super é uma aplicação fullstack de delivery construída com foco em experiência de usuário, identidade visual forte e funcionalidades reais de operação.  
O projeto combina uma interface premium inspirada em marca noturna/tropical com um backend em Node.js, autenticação JWT, banco PostgreSQL e painel administrativo completo.

Clientes podem navegar pelo cardápio, montar açaí personalizado, usar cupons, finalizar pedidos e acompanhar status. Administradores podem gerenciar pedidos, produtos, estoque, cupons, clientes e relatórios.

## Problema resolvido

Pequenos negócios de alimentação precisam de uma experiência digital própria, bonita e funcional, sem depender apenas de marketplaces.  
O projeto simula uma plataforma própria de delivery para uma marca premium de açaí.
