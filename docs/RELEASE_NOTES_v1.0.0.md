# Açaí Lunar S Plus v1.0.0 — Official Stable Release

## Resumo

Primeira versão estável do Açaí Lunar S+ Super, uma plataforma fullstack premium de delivery de açaí com frontend responsivo, backend Express, autenticação JWT, PostgreSQL, Prisma, área do cliente, painel administrativo, checkout real, cupons, relatórios e documentação profissional.
