# PORTFOLIO_PRESENTATION.md

## Açaí Lunar S+ Super — Apresentação para portfólio

## Pitch curto

Plataforma fullstack premium de delivery de açaí com cardápio, carrinho, checkout real, área do cliente, painel administrativo, relatórios, autenticação JWT, PostgreSQL/Neon, Prisma e deploy no Render.

## Pitch comercial

O Açaí Lunar S+ Super simula uma marca premium de delivery de açaí, com experiência visual refinada, compra guiada, combos comerciais, área do cliente e gestão interna para operação da loja.

## Pitch técnico

Sistema fullstack construído com Node.js, Express, PostgreSQL, Prisma e autenticação JWT. O projeto possui API REST, checkout real, persistência em banco, painel administrativo, relatórios operacionais, validações de produção, deploy no Render e banco Neon.
