## Como capturar

1. Abra o deploy:
   https://acai-lunar-s-plus.onrender.com

2. Use zoom 100%.

3. Capture as telas com:
   - Windows: Win + Shift + S
   - Navegador: DevTools para mobile
   - Extensão de screenshot full page, se preferir

4. Salve os arquivos nesta pasta seguindo os nomes acima.
