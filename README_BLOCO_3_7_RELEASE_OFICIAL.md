# Açaí Lunar S+ Super — Bloco 3.7 Release Oficial v1.0.0

## Objetivo

Preparar o projeto para sua primeira release oficial estável no GitHub.

## Entregas

```txt
RELEASE_NOTES_v1.0.0.md
CHECKLIST_RELEASE_v1.0.0.md
docs/RELEASE_GUIDE.md
README_BLOCO_3_7_RELEASE_OFICIAL.md
```

## O que fazer depois de copiar este bloco

Na pasta principal do Git:

```powershell
git add .
git commit -m "release: prepare official v1.0.0"
git push
```

Depois criar a tag:

```powershell
git tag -a v1.0.0 -m "Açaí Lunar S+ Super official stable release"
git push origin v1.0.0
```

Em seguida, criar a release no GitHub usando o conteúdo de:

```txt
RELEASE_NOTES_v1.0.0.md
```
