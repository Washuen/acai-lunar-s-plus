# Açaí Lunar S+ Super v1.0.0 — Official Stable Release

## 🌙 Visão geral

Esta é a primeira versão oficial estável do **Açaí Lunar S+ Super**, uma plataforma fullstack premium de delivery de açaí com experiência visual própria, checkout real, área do cliente, painel administrativo, relatórios, autenticação JWT, PostgreSQL/Neon, Prisma e deploy no Render.
